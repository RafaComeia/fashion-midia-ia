## Story
**As a** visitante da plataforma,
**I want** me cadastrar e fazer login com e-mail/senha ou conta Google,
**so that** possa acessar meu workspace com segurança e ter minha sessão persistida.

## Acceptance Criteria
1. Cadastro via e-mail/senha funcional com validação: e-mail válido, senha mínima 8 caracteres, campos obrigatórios
2. Login via e-mail/senha funcional retornando JWT válido e sessão Supabase
3. Login via OAuth Google funcional com redirect correto após autenticação
4. Sessão persistida com refresh token automático via Supabase Auth (sem logout ao fechar e reabrir o browser)
5. Tela de "Esqueci minha senha" com envio de e-mail de recuperação via Supabase Auth
6. Após primeiro login (novo usuário), redirecionar para `/onboarding`
7. Após login de usuário existente com workspace, redirecionar para `/dashboard`
8. Middleware Next.js protege todas as rotas do grupo `(dashboard)` — redireciona para `/login` se não autenticado
9. Endpoint `POST /auth/sync` no backend sincroniza sessão Supabase e retorna perfil do usuário

## Dev Notes

### Fluxo de autenticação
```
Usuário → /login → Supabase Auth
                        │
              ┌─────────┴──────────┐
              │ email/senha        │ Google OAuth
              ▼                    ▼
         signInWithPassword    signInWithOAuth
              │                    │
              └─────────┬──────────┘
                        │ JWT + refresh token (cookie)
                        ▼
                  middleware.ts (verifica sessão)
                        │
              ┌─────────┴──────────┐
              │ tem workspace_id?  │
              ▼                    ▼
          /dashboard          /onboarding
```

